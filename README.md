# Чит-меню для Stardew Valley
## Описание
Чит-меню для Stardew Valley представляет собой встроенное в игру меню со следующими читами:
1. Чит на деньги
⋅⋅⋅Позволяет получить любое количество монет, которое укажет игрок.
3. Восстановить энергию
---Позволяет восстановить энергию до максимума.
4. Бессмертие
---Режим "Immortality" делает игрока неуязвимым к урону.
5. Убийство одним ударом
---Режим "One Hit Kill" позволяет игроку убивать монстров одним ударом.
6. Заморозка времени
---Остнаовка внутриигрового времени.
7. Изменить размер инвентаря
---Позволяет увеличить или уменьшить размер инвентаря игрока. Для этого необходимо ввеси размер желаемого инвентаря.
8. Изменить скорость
---Позволяет увеличить скорость ('Faster'), уменьшить ('Slower'), установить изначальную скорость персонажа ('Normal').
9. Автополив грядок
---Автоматически поливает все грядки, на которых что-то растет.
10. Вырастить весь урожай
---Моменталньо выращивает все, что посажено на грядках.
